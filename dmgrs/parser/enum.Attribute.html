Expand description

Attributes for the program or for a specific fn or static.

  • Program attributes are written using #![words here]

Variants

Call(Spanned<&'static str>, Vec<Attribute>)

Call attributes are like foo(bar), foo(), or even foo

The current parser can’t distingush between foo() (with parens) and foo (no parens). This is suspected to not ever be important.

Eq(Spanned<&'static str>, Spanned<&'static str>)

Eq attributes are like foo = "bar".
